# Project overview
A modern, responsive, and visually appealing landing page built using **HTML5**, **CSS3**, and **JavaScript**. This project showcases a digital agency website with smooth navigation, dark mode functionality, responsive design, and an engaging hero section with a background video.

---

## 📌 Features

- 🎨 Modern and clean UI design
- 📱 Fully responsive layout
- 🌙 Dark mode toggle
- 🍔 Mobile-friendly hamburger navigation
- 🎥 Hero section with background video
- ⚡ Smooth scrolling navigation
- ✨ Interactive hover animations
- 📊 Statistics section
- 💼 Services showcase
- 📧 Contact CTA section
- 🔗 Social media icons using Font Awesome
